# 波形格式说明与预设库

## 格式说明

波形通过 `pulse-x:[...]` 指令下发，`x` 为通道（`a` 或 `b`）。  
每帧为一个 16 进制字符串，服务端每 100ms 自动消耗一帧并循环，最多 4 帧/次发送。

详细字节含义见 [DG-LAB 官方协议文档](https://github.com/DG-LAB-OPENSOURCE/DG-LAB-OPENSOURCE)。

---

## 内置预设

可直接复制帧字符串填入指令：

| 名称 | HEX 帧 | 特征 |
|------|--------|------|
| 连续稳定 | `1E9082B46464646464646464646464640000FF00` | 持续均匀输出，无变化 |
| 脉冲节奏 | `14508000000000000000000000000000FFFFFF00` | 短促跳动，节奏感强 |
| 呼吸渐变 | `145082B41414141414141414141414140000FF00` | 强度周期性渐强渐弱 |
| 强力冲击 | `3C9082B43C3C3C3C3C3C3C3C3C3C3C3C3CFFFFFF` | 高强度持续输出 |

---

## 使用示例

```
# 单帧重复发送（最常用）
pulse-a:["1E9082B46464646464646464646464640000FF00","1E9082B46464646464646464646464640000FF00"]

# 多帧循环（4帧呼吸效果）
pulse-a:["0E5082B40E0E0E0E0E0E0E0E0E0E0E0E0000FF00","1E5082B41E1E1E1E1E1E1E1E1E1E1E1E0000FF00","2E5082B42E2E2E2E2E2E2E2E2E2E2E2E0000FF00","1E5082B41E1E1E1E1E1E1E1E1E1E1E1E0000FF00"]
```
