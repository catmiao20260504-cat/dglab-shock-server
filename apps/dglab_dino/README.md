# 小恐龙跑酷 × DG-Lab 电击联动

赛博朋克风格的像素跑酷游戏。每次撞到障碍物死亡，自动向郊狼设备发送电击指令。强度、持续时间、波形均可在右侧面板实时调整。

## 功能

- 像素恐龙跑酷（空格 / 点击跳跃，↓ 蹲伏）
- 仙人掌 + 翼龙双障碍，随分数自动加速
- 死亡触发双通道电击（默认 A=20 B=10，持续 1500ms）
- 右侧控制面板：强度滑块、±5 微调、持续时间设置
- 6 种内置波形 + 自定义 HEX 导入
- 手动测试电击按钮
- 紧急停止
- 赛博朋克霓虹视觉风格

## 使用方法

1. 启动服务端：`python server/dglab_server.py`
2. 郊狼 App 扫码绑定
3. 浏览器打开 `apps/dino-game/index.html`
4. 点击右侧「接入测试 / 连接」，等待状态显示「已绑定」
5. 按空格或点击画布开始游戏

## 电击触发条件

| 触发事件 | A 通道强度 | B 通道强度 | 持续时间 |
|----------|-----------|-----------|----------|
| 恐龙死亡 | 面板可调（默认 20） | 面板可调（默认 10） | 面板可调（默认 1500ms） |

## 依赖

- `dglab_server.py` v1.0+
- 接入方式：`POST /cmd` + `GET /status`（HTTP API）
- 无其他外部依赖，单文件 HTML
