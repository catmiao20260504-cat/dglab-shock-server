# 贪吃蛇 × DG-Lab 电击惩罚版

赛博朋克风格的贪吃蛇游戏。撞墙或咬到自身死亡时，自动向郊狼设备发送双通道电击惩罚。强度、时长、波形均可在右侧面板实时调整。

## 功能

- 经典贪吃蛇玩法（方向键控制）
- 画布 700×500，网格 35×25
- 吃到食物得分，速度随分数自动提升（初始 150ms/格，最快 70ms/格）
- 死亡（撞墙 / 咬自身）触发双通道电击
- 右侧控制面板：
  - A / B 通道强度滑块（0~200）及 ±5 微调
  - 电击持续时间设置（默认 1600ms）
  - 6 种内置波形预设 + 自定义 HEX 导入
  - 手动测试电击按钮
  - 设备实时强度显示
  - 紧急停止
- 历史最高分本地记录（localStorage）

## 使用方法

1. 启动服务端：`python server/dglab_server.py`
2. 郊狼 App 扫码绑定
3. 浏览器直接打开 `index.html`（无需本地服务器）
4. 右侧面板填写服务端地址（默认 `http://localhost:5679`），点击「连接测试」
5. 状态显示「已绑定」后，按方向键开始游戏

## 电击触发条件

| 触发事件 | A 通道强度 | B 通道强度 | 持续时间 |
|----------|-----------|-----------|----------|
| 撞墙死亡 | 面板可调（默认 20） | 面板可调（默认 10） | 面板可调（默认 1600ms） |
| 咬自身死亡 | 同上 | 同上 | 同上 |

未连接设备时游戏正常运行，死亡不触发电击。

## 依赖

- `dglab_server.py` v1.0+
- 接入方式：`POST /cmd` + `GET /status`（HTTP API）
- 无其他外部依赖，单文件 HTML，直接双击可运行
