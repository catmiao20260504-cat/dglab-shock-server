# 通用控制面板

连接 DG-Lab 郊狼设备后，手动调节双通道强度、选择波形、发送原始指令，并查看实时日志。适合调试和测试使用。

## 功能

- A / B 双通道强度滑块（0~200），支持 ±1 / ±10 微调
- 6 种内置波形预设一键切换
- 自定义波形 HEX 导入
- 原始指令输入框（直接发 message 字段内容）
- 实时消息日志
- 紧急停止按钮

## 使用方法

1. 启动服务端：`python server/dglab_server.py`
2. 郊狼 App 扫码绑定
3. 浏览器打开 `apps/control-panel/index.html`
4. 填写服务端地址（默认 `http://localhost:5679`），点击「接入测试 / 连接」
5. 状态显示「已绑定」后即可操作

## 电击触发条件

本应用为手动控制面板，无自动触发，所有操作均由用户主动发起。

## 依赖

- `dglab_server.py` v1.0+
- 接入方式：`POST /cmd` + `GET /status`（HTTP API）
